## 🧩 About Me

20년 가까이 게임을 즐겨온 **코어 게이머**로서, 게임의 본질적인 재미와 공정성을 해치는 부정 행위를 **절대 용납할 수 없다는 신념**을 가지고 있습니다.  
이러한 문제의 근원을 해결하고자 **정보보안 분야**에 뛰어들었으며, 복잡한 시스템의 무결성을 보호하고 잠재적 위협을 예측하는 **고도의 분석적 사고**를 길러왔습니다.

> **목표:**  
> 게임의 '재미(Fun)'와 '사용자 만족'을 보장하는 FQA 전문가로 성장하여,  
> 모든 게이머가 신뢰할 수 있는 환경을 만드는 데 기여하는 것.

## 🎯 주요 관심 분야 (Security Research)

### 📡 ESP (Extra Sensory Perception)
- **내용:** 비정상적인 위치/아이템 표시 기능
- **분석:** 게임 메모리 구조 분석, 네트워크 패킷 추적  
- **대응:** 오버레이 탐지, 화면 데이터 변경 감지

---

### 🎯 AIMBOT (자동 조준)
- **내용:** 비정상 조준 자동화 치트
- **분석:** 마우스 이벤트 분석, 픽셀 기반 조준 탐지  
- **대응:** AI 기반 비정상 패턴 인식

---

### 🛠 메모리 변조 (Memory Hacking)
- **내용:** 메모리 값을 조작해 게임 밸런스 붕괴
- **분석:** Cheat Engine 탐지, 무결성 검사  
- **대응:** ASLR, 메모리 암호화

---

### 🌐 서버 정보 탈취
- **내용:** 패킷 스니핑을 통한 서버 구조 분석  
- **분석:** 세션 인증, 암호화 필요성 강조  
- **대응:** SSL/TLS 강화, 패킷 요청 패턴 분석

---

### 🔄 리버싱 (Reversing)
- **내용:** 코드 분석을 통한 치트 제작
- **분석:** 디컴파일링, 난독화 및 방어 기술 연구  
- **대응:** Anti-Debugging, 실행 시간 조작 탐지

---

### 🕵️ Ghidra
- **활용:** 오픈소스 리버스 엔지니어링 툴  
- **내용:** 게임 바이너리 분석, 플러그인 개발로 치트 탐지 역량 강화

---

### 👩 IDA Pro
- **활용:** 상용 리버싱 도구  
- **내용:** IDAPython 활용 자동 분석 및 탐지 스크립트 개발

---

### 🎮 언리얼 엔진 (Unreal Engine)
- **활용:** C++, 블루프린트로 게임 시스템 분석 및 개발  
- **연구:** 리플리케이션, 메모리 무결성 검증, 비정상 접근 탐지 로직 개발
